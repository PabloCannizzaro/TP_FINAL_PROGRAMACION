Colocá aquí las imágenes de las cartas para el frente y el dorso.

Nombres esperados (minúsculas):
- Reverso por defecto: back.png
- Frente: <rank>_of_<suit>.png
  donde <rank> es uno de: ace, 2, 3, 4, 5, 6, 7, 8, 9, 10, jack, queen, king
  y <suit> es uno de: hearts, diamonds, clubs, spades

Ejemplos:
- ace_of_hearts.png
- 10_of_spades.png
- queen_of_diamonds.png
- back.png

Si falta algún archivo, el frontend genera un SVG de placeholder automáticamente.
